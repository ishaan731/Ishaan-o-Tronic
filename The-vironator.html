The Vironator

A project to save us all from COVID-19

I have come up with 'The Vironator' that

dispenses the sanitizing liquid

automatically, without any direct

contact with dispenser. This mechanism

would assist in preventing infections

from spreading in any place. The

sensor would sense the hand

and would drop some sanitizing

liquid on the hand. It is a very

useful application, especially in the current pandemic

which is is highly contagious.
