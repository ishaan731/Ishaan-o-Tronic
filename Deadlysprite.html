Deadly Sprite

A truly amazing game for children !

Hello, friends, this is my new project called

deadly sprite. It mainly uses a button, an LCD board,

and a micro-controller like an Arduino. So when we

press the button, the sprite would jump over the

bricks and would come down when the

bricks are over. There is also a

score board to make the game more challenging.

It is not an only exciting game but it also

works on the focus levels of children.
